Abstract

(Simulation of Guitar with MATLAB) My project is to simulate guitar's strings sound and synthesis strings sound using matlab .

Problem Specification

The main task of this project is using matlab to play a guitar's song and developing Graphical user interface to Simulate the vibrating
string with possibility to set different parameters include initial positions, velocities, distributed friction. It is possible to plot graph,
spectrogram, animated graph, listen stereo sound and realtime.

Results
